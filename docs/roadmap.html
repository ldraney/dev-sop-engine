Architecture User Stories Roadmap

Roadmap

Current v0.1.0 - MVP

Minimal working generator

  • CLI: npx dev-sop-engine .
  • Read sop.json from current directory
  • Generate .claude/settings.json
  • Generate .claude/CLAUDE.md from memory config
  • Copy hooks from sop/hooks/
  • Generate .mcp.json
  • Dogfood on this repo

Planned v0.2.0 - Skills & Agents

Full Claude feature support

  • Generate .claude/skills/ from skills config
  • Generate .claude/agents/ from agents config
  • Support inline content and file references
  • JSON Schema for sop.json validation

Planned v0.3.0 - Init & DX

Developer experience improvements

  • CLI: npx dev-sop-engine init - scaffold starter config
  • CLI: npx dev-sop-engine validate - check sop.json
  • Better error messages with line numbers
  • TypeScript types published for sop.json

Future v0.4.0 - Advanced Features

Power user features

  • CLI: npx dev-sop-engine diff - preview changes
  • CLI: npx dev-sop-engine watch - auto-regenerate
  • Generate from remote sop.json URL
  • Scaffold templates (--scaffold flag)

Future v1.0.0 - Stable

Production ready

  • Stable sop.json schema (no breaking changes)
  • Comprehensive documentation
  • CI/CD examples
  • Published to npm

Out of Scope

  • GUI / web interface
  • Plugin system (reconsider post-1.0)
  • MCP server mode (CLI-only for now)
  • Cloud sync / remote config management
