Roadmap

Done v0.1.0 - Core Generator

Working generator with full feature coverage

  • CLI: npx dev-sop-engine .
  • Generate .claude/settings.json with hook wiring for all 11 events
  • Rules with hook-based validators (PreToolUse matchers, conditions)
  • Structured event logging with custom handlers
  • Skills generation (content_file → SKILL.md)
  • Agents generation (prompt_file → YAML frontmatter)
  • Generate .mcp.json with smart merge (preserves manual servers)
  • MCP server mode (sop_generate tool)
  • Subagent rule inheritance
  • Dogfood on this repo (3 rules, 2 skills, 2 agents)

Current v0.2.0 - Adoptability

Make it usable by someone other than the author

  • Add README.md
  • Align documentation with actual implementation
  • CLI: npx dev-sop-engine init — scaffold starter config
  • Basic input validation and error messages for malformed sop.json
  • Publish to npm
  • Set version to 0.x (currently incorrectly at 1.0.0)

Planned v0.3.0 - Reliability

Testing and validation

  • JSON Schema for sop.json validation
  • CLI: npx dev-sop-engine validate — check sop.json
  • Unit tests for generator logic
  • GitHub Actions CI
  • Better error messages with line numbers

Future v0.4.0 - Advanced Features

Power user features

  • CLI: npx dev-sop-engine diff — preview changes before generating
  • CLI: npx dev-sop-engine watch — auto-regenerate on sop/ changes
  • Shareable config presets (import/extend another sop.json)
  • Dry-run mode

Future v1.0.0 - Stable

Production ready

  • Stable sop.json schema (no breaking changes)
  • Comprehensive documentation
  • CI/CD pipeline examples for teams
  • TypeScript types published for sop.json

Out of Scope

  • GUI / web interface
  • Plugin system (reconsider post-1.0)
  • Cloud sync / remote config management
